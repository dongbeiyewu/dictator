# dictator
一款动作游戏
## 故事背景
+ 古代的暴君为了长生不老让魔法师寻找强大的魔法
+ 邪恶的恐怖组织为了统治世界让科学家制造武器
+ 时空交错，暴君获得了强大给武器。恐怖组织获得了魔法
+ 主角为了打败他们踏上征途
## 游戏类型
  + 第三人称视角动作游戏
